emsg-sdk-android
================

emsg sdk for android

工程目录介绍：

EmsLibrary为ems开源sdk工程 可作为library引用库使用

EmsgClient为基于EmsLibrary使用demo




友情提示：

若sdk提供的接口方法足够使用，用可直接参照EmsgClient 相关使用方法进行业务扩展即可

若sdk提供的接口方法无法满足使用场景时,可对EmsLibary增加相关接口回调,或者直接联系我们.
